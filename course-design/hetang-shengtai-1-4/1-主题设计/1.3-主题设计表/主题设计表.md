# 主题设计表

> 版本:第 1 版
> 日期:2026-06-02
> 修订原因:步骤一主题方向荷塘生态修复已确认，建立权威源文件
| 项目 | 内容 |
|---|---|
| **主题背景** | 荷塘等浅水水体因氮磷过量输入引发蓝藻水华，导致水体发黑发臭、鱼虾死亡、荷花无法正常生长。水体富营养化是全球性环境问题，也是学生身边可感知的真实生态挑战。 |
| **挑战任务** | 学生扮演"小小生态修复工程师"，为受蓝藻困扰的荷塘制定生态修复行动方案，服务对象为公园管理方或社区居民。 |
| **学习成果概要** | |
| - 学习成果名称 | 荷塘生态修复行动方案（含水质数据分析与修复措施清单） |
| - 学习成果图片 | 暂无实物图；成果形态：含数据分析表与修复措施清单的书面方案 |
| **知识概要** | |
| - 学科核心知识 | 科学：蓝藻特征、水体富营养化成因、生态链；数学：数据收集与统计分析；综合实践活动：实地调研与问题解决流程 |
| - 问题解决的过程与方法 | 提出问题→收集水质数据→数据分析与归因→制定修复方案→评估可行性→呈现方案 |
| - 核心素养 | 科学探究、数据思维、责任担当 |
