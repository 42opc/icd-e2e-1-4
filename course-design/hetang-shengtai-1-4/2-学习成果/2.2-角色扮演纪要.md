# 2.2 角色扮演纪要 — 过程记录

> 版本:第 1 版
> 日期:2026-06-02
> 修订原因:角色扮演采访完成，汇总三轮 persona 回写结果
## 采访一：课程学员视角

> 受众角色：小学高年级学生（课程学习者）
> 来源：1.3 主题设计表目标受众字段

### 问题一：最终成果期望形式

- 期望形式：**有据可查的书面行动方案**
- 方案构成：
  - 第一块：水质数据分析表（自行测量、统计整理的真实数据）
  - 第二块：修复措施清单（根据数据分析得出的步骤和方法）
- 格式参考正式工作方案，公园管理方拿到后能照着执行
- 引用：1.3 主题设计表 — 学习成果概要（书面方案含数据分析表与修复措施清单）

### 问题二：评判角度

- **科学准确性** — 蓝藻特征、富营养化成因、生态链知识是否正确
  - 引用：1.3 主题设计表 — 科学：蓝藻特征、水体富营养化成因、生态链
- **数据质量** — 是否真实收集水质数据，用统计方法分析，从数据推出原因
  - 引用：1.3 主题设计表 — 数学：数据收集与统计分析
- **方案可行性** — 修复措施是否有科学依据，荷塘现场能否实施
  - 引用：1.3 主题设计表 — 综合实践活动：实地调研与问题解决流程
- **流程完整性** — 是否按"提出问题 → 收集数据 → 分析归因 → 制定方案 → 评估可行性 → 呈现方案"走完
  - 引用：1.3 主题设计表 — 问题解决的过程与方法

### 问题三：评分与加分机制

数据分析维度：
- 测量 3 种及以上水质指标（氮、磷、pH、透明度）且每项有 3 次以上记录 → 满分
- 仅 1 种指标或无实测数据、只查资料 → 扣分

修复措施维度：
- 每条措施写明对应科学原理 → 满分
- 措施有但无原理说明 → 部分得分

加分情形：
- 提出后续**监测计划**（如每月重测水质，对比修复效果）
- 考虑修复措施的**成本与可操作性**，给出优先级排序

备注：案例库中无与"学生作为成果接收者"直接对应的加分示例。以上来源：1.3 主题设计表知识概要核心素养映射推导，非脑补。

## 采访二：产品接收者视角

> 受众角色：公园管理方（成果最终使用方）
> 来源：1.3 主题设计表挑战任务服务对象

### 问题一：最终成果期望形式

- 期望：**可直接用于开展后续工作的书面行动方案**
- 方案构成：
  - 水质数据分析表（含测量值、图表、问题严重程度说明）
  - 修复措施清单（逐条说明操作步骤和理由）
- 格式要求：有封面和目录，不超过 10 页，便于存档和传阅
- 引用：1.3 主题设计表 — 学习成果名称（荷塘生态修复行动方案含数据分析表与修复措施清单）

### 问题二：评判角度

- **数据可信度** — 数据来自真实测量（多次、多指标），非网络参考值
- **原因分析** — 能清楚说明荷塘变差的原因，便于向上级汇报
  - 引用：1.3 主题设计表 — 学科核心知识：水体富营养化成因
- **措施可落地** — 每条措施说明具体做法和所需资源，可交施工方或申请经费
- **逻辑完整** — 从发现问题到分析原因到制定措施，有完整推进路径
  - 引用：1.3 主题设计表 — 问题解决的过程与方法

### 问题三：评分与加分机制

数据分析部分：
- 测量 3 种及以上关键水质指标（透明度、总氮、总磷、pH 至少 3 项），且标注测量时间和方法 → 满分
- 仅 1 种指标或数据来源为网络资料（非现场实测） → 大幅扣分

修复措施部分：
- 每条措施写明"做什么、为什么这样做、大致怎么做" → 满分
- 仅列原则性表述（如"应减少氮磷排放"）无具体步骤 → 部分得分

加分情形：
- 对措施进行**优先级排序**并给出判断依据，减少管理方决策难度
- 含**后续监测建议**（如每月复测，对比修复前后数据）
- 提供**实施条件评估**（哪些措施需专业设备，哪些学校自己能完成）

备注：案例库中无以"公园管理方"为受众的目标受众条目。以上来源：1.3 主题设计表问题解决流程、2.1 名称与类型分析可行性要求，映射推导，非脑补。

## 学科可达性调整

### 可达性判定

| 知识点 | 可达性 | 简要依据 |
|---|---|---|
| 科学：蓝藻特征 | 部分可达 | 5~6 年级课标仅列酵母菌、霉菌、病毒为例，蓝藻不在列；藻类分类要求在 7~9 年级 |
| 科学：水体富营养化成因 | 部分可达 | 5~6 年级课标以项目研究方式涉及水体污染调查，但氮磷循环机制未作内容要求 |
| 科学：生态链 | 可达 | 5~6 年级课标明确要求说出常见动物和植物之间吃与被吃的链状关系 |
| 数学：数据收集与统计分析 | 可达 | 数学第三学段统计与概率数据收集整理与表达明确列入 |
| 综合实践活动：实地调研 | 部分可达（来源不完整） | 本地数据包无综合实践活动独立课标文件 |

### 课标对齐情况

| 知识点 | 课标依据 | 对齐结论 |
|---|---|---|
| 蓝藻特征 | 科学 5~6 年级 §5.2（微生物列举：酵母菌、霉菌、病毒，不含蓝藻）；藻类特征属 7~9 年级 §5.2④ | 超出本学段范围；可作情境素材，不作达标知识要求 |
| 水体富营养化成因 | 科学 5~6 年级 §7 学习活动建议（探究水体污染原因，过程性活动，非掌握机制） | 知识落点须降至认识人类活动对水体的影响，不要求掌握氮磷循环机制 |
| 生态链 | 科学 5~6 年级 §5.6⑥（说出常见动植物吃与被吃的链状关系，基础食物链完全在范围内） | 可达；须限定为简单链状关系，不涉及食物网结构（食物网为 7~9 年级要求） |
| 数据收集与统计分析 | 数学第三学段 §统计与概率（数据收集整理表达；折线图、条形图） | 完全在范围内，是本学段明确要求 |
| 综合实践活动：实地调研 | 本地数据包无独立课标文件；可参照科学课标 5~6 年级 §7 学习活动建议作间接依据 | 课标来源不完整；建议补充确认或改标注为科学（探究实践） |

### 学科调整建议

- **调整一（必要）— 蓝藻特征表述降层**
  - 原表述：蓝藻特征
  - 调整为：以蓝藻为情境素材，认识微生物对水体的影响
  - 知识点从"蓝藻特征描述"降至"知道水华现象与微生物有关"
  - 对应课标：5~6 年级 §5.2 列举常见微生物

- **调整二（必要）— 水体富营养化成因范围界定**
  - 原表述：水体富营养化成因（含氮磷过量输入）
  - 调整为：通过调查认识水体变差的主要原因（如过度施肥、生活污水）
  - 不要求掌握氮磷循环化学机制
  - 对应课标：5~6 年级 §7 探究水体污染原因

- **调整三（必要）— 生态链限定为基础食物链**
  - 原表述：生态链
  - 调整为：荷塘基础食物链（如水草 → 鱼虾 → 荷塘生物）
  - 明确不涉及食物网结构
  - 对应课标：5~6 年级 §5.6⑥

- **调整四（建议）— 综合实践活动学科归属补充**
  - 建议设计师补充综合实践活动课标引用
  - 或将学科归属改标注为科学（探究实践），说明暂无独立课标覆盖
