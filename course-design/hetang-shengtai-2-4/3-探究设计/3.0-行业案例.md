## 一、行业案例（共 6 条，含 1 项现行行业标准）

### 案例 1｜重庆开州汉丰湖小微湿地群 — 消落带基塘荷花修复工程（2024 入选全国美丽河湖）

**场景：** 国家湿地公园消落带退化，挖塘种荷，修复 6.77 公顷坑塘水面

**核心做法：**
- 在消落带缓坡挖数千个水塘，种植荷花、美人蕉、菱角等耐水淹水生植物共 1100 亩
- 塘基栽植落羽杉乔木 500 余株，形成"水中林泽 + 景观基塘"复合系统
- 5 大技术体系：基塘工程 / 林泽工程 / 生态浮床 / 鸟类生境 / 多功能缓冲系统

**水质/生态成果：** 消落带物种显著恢复；入选重庆首届生态保护修复十大案例、全国美丽河湖优秀案例

**设计判断：** 荷花塘净化氮磷的生态机制在本主题可直接迁移；水位动态设计方面，校园静水荷塘场景更简单（无季节性水位变化），适配性更强

> 来源：https://m.12371.gov.cn/content/2024-03/28/content_460989.html
> 补充：https://h5.cqliving.com/info/detail/76496974.html

### 案例 2｜湖南常德市堰塘型黑臭水体综合治理 — 莲塘修复（2023 省级十佳案例）

**场景：** 农村社区堰塘长期承接生活废水，严重黑臭，居民投诉

**核心做法：**
- 控源截污 → 清淤疏浚 → 生态修复三步走
- 分散式三级净化池 + 人工湿地处理生活废水
- 植物配置：鸢尾花 + 菖蒲阶梯搭配，净化氮磷

**水质成果：** 水体透明见底，溶解氧大幅提升，居民满意度显著改善

**设计判断（关键约束）：** 截污优先是必要前置条件——仍有活跃污水排入的荷塘必须先截污再修复，否则植物净化效果无法维持。这是向校方汇报可行性时必须说清的约束

> 来源：https://sthjt.hunan.gov.cn/sthjt/ztzl/wrfz2018/202311/t20231116_32406825.html
> 补充：https://www.changde.gov.cn/cdzx/cdyw/content_1093529

### 案例 3｜辽宁公安司法管理干部学院海绵校园 — 校园人工湿地荷花池水质净化（有量化数据）

**场景：** 校园内 25 级生态叠水堰，总长 600 m、面积 11500 m²，再生水回用湿地

**核心做法：**
- 植被净化系统：挺水植物（荷花）+ 沉水植物 + 浮水植物分区配置
- 梯级湿地：砾石过滤 → 植物净化 → 梯级跌水 → 营养物去除

**量化水质净化数据：**

| 指标 | 去除率 |
|---|---|
| 总磷（TP） | 74.9% |
| 氨氮（NH₃-N） | 61% |
| COD | 41.18% |
| 悬浮物（SS） | 64.71% |

**设计判断：** 校园场景可控污染源 + 固定用水量是最强适配场景；**TP -74.9%、氨氮 -61% 是向校方/居民汇报可行性时可直接引用的硬性数据**

> 来源（工程案例）：https://www.turenscape.com/project/detail/4907.html
> 来源（学术论文，含量化数据）：https://www.ecologica.cn/html/2019/16/stxb201904020640.htm
> 查询时间：2026-06-03

### 案例 5｜昆山排水管网智慧运营平台 — 水质监测仪表盘（2024 全国典型案例）

**场景：** 市政排水系统，物联网 + 大数据可视化仪表盘，水质实时监测与历史对比

**仪表盘功能模块：**
- 领导驾驶舱 / 智慧看板
- 物联网在线监测 + 水质预警
- "数据时光机"：历史水质变化可视化（Before-After）

**成果：** 投诉件减少 50%；养护费用节约 1/3

**设计判断：** "数据时光机"（Before-After 可视化）+ 当前状态灯是向社区居民和校方展示修复成效的最直观仪表盘形式。该平台为市政级投入，荷塘监测仪表盘可降级为低成本 IoT 传感器 + 开源可视化工具（如 Grafana / 微信小程序）

> 来源：https://www.suzhou.gov.cn/szsrmzf/qxkx/202412/9bdff6fbf3c54d29b8f33a1e04b8b663.shtml

## 二、行业 Persona — 水生态修复工程师视角

> 来源：沈博·R 角色扮演模拟（comment 26ac6a91，基于 WebSearch 真实案例）

**真实工作场景：** 接社区堰塘/荷花塘类小微水体治理项目，甲方（街道办、物业或学校总务处）要求"修好塘 + 能说清楚修复效果"。项目难点不是技术，而是**如何在小预算内交出可量化成果，并向非专业的居民和校方管理层说清楚"为什么这个方案有效、长期能维持"**。

**真实工作五步：**

| 步骤 | 内容 | 关键判断点 |
|---|---|---|
| Step 1 现场诊断 | 采水样检测 pH/DO/氨氮/总磷/透明度；调查底泥、进出水口、周边污染源 | 不先做方案，先搞清楚"这口塘为什么臭" |
| Step 2 截污 + 清淤 | 封堵活跃污水入口；机械清淤去除富磷厌氧底泥 | 跳过截污直接种植，一个雨季后回到原点（常德案例教训） |
| Step 3 水生植物分层配置 | 挺水层（荷花+鸢尾/菖蒲）+ 浮水层（睡莲）+ 沉水层（苦草/黑藻） | TP -74.9%、氨氮 -61%（辽宁校园量化数据） |
| Step 4 监测与仪表盘 | IoT 传感器 + Before-After 折线图 + 实时状态灯（绿/黄/红） | 昆山"数据时光机"是成熟参考原型，可低成本复刻 |
| Step 5 社区/校方汇报 | 诊断结论 → 方案逻辑 → 标准合规性 → 预期指标 → 长效监测演示 | 居民最关心"多久不臭"和"以后谁维护" |

**行业交付物清单（直接映射到学生挑战任务）：**

| 交付物 | 描述 | 课程对应 |
|---|---|---|
| 水质检测报告（修复前/后） | pH / DO / 氨氮 / TP / 透明度 5 项指标 | 科学：采样与数据分析 |
| 植物配置施工图 | 挺水/浮水/沉水分层种植平面图 | 科学：生态修复方案 |
| 长效监测方案 | 传感器布点、监测频率、预警阈值 | 数学：数据统计与可视化 |
| 水质仪表盘（面向公众） | Before-After + 实时状态灯 | 综合实践：成果汇报 |
| 社区/校方汇报材料 | 含合规依据、预期指标、维护责任主体 | 综合实践：方案论证 |

**成功评判标准（行业视角，可直接用作学生评估维度）：**
- 水质指标达标：氨氮 ≤ 1.0 mg/L、TP ≤ 0.1 mg/L（地表水 III 类）
- 透明度恢复：肉眼可见透明度 ≥ 0.5 m（居民感知层最直观）
- 荷花恢复率：覆盖率达设计面积 70%（第二生长季评估）
- 无反弹：修复后 12 个月监测无明显恶化
- 居民/校方认可：汇报后甲方能复述"为什么不会反弹"

## 三、综合研究结论

**可落地性核心结论：**
1. 荷花塘生态修复最优路径：**截污优先 → 清淤 → 水生植物多层配置（荷花+沉水+挺水）→ 生态监测**
2. 量化可行性论据（案例 3）：TP 可降低 75%、氨氮降低 61%，直接可引用支撑校方汇报
3. 监测仪表盘最小可行版本：低成本 IoT 传感器 + Before-After 折线图 + 状态灯（可用开源工具实现）
4. 合规框架：LY/T 3353-2023 提供直接引用依据，显著提升方案公信力
